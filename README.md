        ```python
        # 변수만 선언, 값만 할당하면 숫자형으로 지정
        # C, C++, Java, C#처럼 형지정이 없음
        val = 10    # 정수형
        val = 3.15  # 실수형

        # 특수 숫자형
        binVal = 0b11111111     # 255
        octVal = 0o11           # 9     1 ~ 7 10(8)
        hexVal = 0xFF           # 255   0 ~ 9ABCDEF
        print('2진수:', binVal, '8진수:', octVal, '16진수:', hexVal)
        ```
        - 문자열(특수형태 리스트)형(연산, 문자열 포맷, 함수)
        ```Python

        ```
        - 리스트형, 튜플형(연산, 함수)
            - 리스트는 수정, 삭제 가능
